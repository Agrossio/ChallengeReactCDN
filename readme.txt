# Setup:
1) Inicializar proyecto Node
2) Instalar live-server para servir los archivos (HTML, favico, etc)
3) Corregir "m" a "M" en la ruta del componente "Module.jsx" del HTML
4) Agregar Script para correr live-server al package.json
5) Babel va a transpilar el jsx a js

# Para editar hacerlo en 
    template/modules/module/Module.jsx

# No editar el INDEX.HTML 

# Los estilos hacerlos mediante CSS en el archivo
    template/modules/module/componente.css
    o bien con estilos en linea

# Hacer todo el desarrollo en un unico archivo para evitar problemas
    template/modules/module/Module.jsx
